What do we need to do to build a feature

  1. Go back to development
  2. Pull the development branch (so the new branch starts up to date)
  3. Checkout a new branch `git checkout -b feat-add-bla`
  4. Build a feature & commit step by step
  5. Push our branch to github
  6. Make a pull request (with a summary, context, why, what, doubts, screenshot)
  7. Ask for review, wait .. patiently, coffee maybe
  8. Implement feedback, or enjoy your work being merged
